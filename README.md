# Rotation_XYZ

Программа позволяет пользователю поворачивать объект(прямоугольник с вписанным в него элипсом) вокруг осей oX, oY и oZ на углы, заданные в радианах. В отдельных окнах отображаются проекции объекта на плоскости oXY, oXZ, oYZ.

Ссылка на собранный .exe файл: https://drive.google.com/drive/folders/1Vec8wHbTsmrdyCdlROXi2oJjkVYGz8BA

Файл называется Triangle.exe и лежит в папке .../debug
